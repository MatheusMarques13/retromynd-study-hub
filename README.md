# RetroMynd Study Hub

> Coding practice & quiz platform with **Comfy** + **Retro** modes.

## Features

- **Quiz Mode** — 45+ knowledge questions on programming, algorithms, AI/ML, and dev culture
- **Coding Mode** — LeetCode-style problems with built-in code editor and test runner
- **History** — Track your scores, streaks, and progress over time
- **Dark Mode** — Full dark theme with twinkling stars and moon
- **Retro Mode** — Toggle the OG retro aesthetic (scanlines, VT323 font, pixel art)
- **Comfy Mode** — Soft pastel notebook aesthetic with Kalam handwriting font

## Tech Stack

- Pure HTML/CSS/JS — zero dependencies, single file
- CodeMirror for the code editor
- localStorage for persistence

## Deploy

Static site — just serve `index.html`. Works on Vercel, Netlify, GitHub Pages, or any static host.
